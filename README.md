## 续费后多久生效？延期从哪天算起？

付款之后立即生效，不需要等待。

但有一个细节：续费延期的起始时间是**原到期日**，不是你付款的当天。

举个例子——你的 VPS 3 月 22 日到期，3 月 26 日才付款续费。续费后的到期时间会是 3 月 22 日 + 一年 = 次年 3 月 22 日，而不是从 3 月 26 日开始算。晚续费几天不会影响总使用时长，但也不会补偿这几天——所以宽限期内越早续费越好。

## 什么情况下搬瓦工无法续费？

有几种情况会导致续费失败或不可续：

**情况一：续费开关被关闭。** 后台 VPS 列表里有 "Renewal: ON/OFF" 状态。如果被误操作关成 OFF，账单到期就不会生成，也无法续费。发现这个问题及时联系客服提交工单，申请重新开启。

**情况二：套餐已被停售且明确不支持续费。** 早期的 OpenVZ 架构套餐已全部停止续费，如果你还持有这类老套餐，到期就消失了，只能重新买 KVM 套餐迁移数据。

**情况三：超过删除时限。** 暂停后长时间没有续费，官方最终会删除 VPS 及所有数据（包括快照和备份）。到了这一步就彻底没机会了，数据也无法恢复。

## FAQ：搬瓦工续费常见问题整理

**Q：搬瓦工续费价格和购买价格一样吗？**

A：是的，完全一样。续费价格锁定在你初次购买时的价格，套餐后来涨价不影响你的续费金额。

**Q：买的时候没用优惠码，续费时能补用吗？**

A：不能。优惠码只在首次购买时生效，并绑定到该套餐的所有后续续费上。没有在购买时用优惠码，就只能按原价续费了。

**Q：搬瓦工支持自动续费吗？**

A：官方不会主动从你的支付宝、信用卡或 PayPal 里扣款。但如果你提前往账户余额里充了足够的钱，到期出账单时系统会自动从余额中扣除，相当于实现了自动续费。

**Q：快到期了但账单还没出，是不是有问题？**

A：正常的。搬瓦工账单在到期当天（美国时间）生成，对应北京时间大概是到期日的次日凌晨。到期之前找不到续费入口是正常现象，等到那天凌晨就会有了。

**Q：过了7天忘续费了，VPS 被暂停还能救吗？**

A：可以。暂停后账单还在，付款之后 VPS 会恢复运行。但如果拖了太久（超过几周）导致 VPS 被删除，数据就彻底没了，无法恢复，也不能再续费开启。
